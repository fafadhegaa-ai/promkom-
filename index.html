Cara Kerja Sistem

Proses check-in yang sederhana namun powerful

1

Login Secure

Mahasiswa login dengan email & password. Sistem mengenkripsi data dan generate JWT token

2

Select Class

Pilih kelas dari daftar yang diikuti. Sistem menampilkan informasi lokasi kelas

3

Enable GPS

Aktifkan GPS di device. Sistem mendapat lokasi real-time user (latitude, longitude)

4

Verify Distance

Hitung jarak menggunakan Haversine formula. Cek apakah user dalam radius 100m

5

Check Result

Jika dalam radius ✅ BERHASIL | Jika di luar radius ❌ DITOLAK

6

Log Activity

Simpan ke database dengan device tracking, timestamp, dan audit log otomatis
